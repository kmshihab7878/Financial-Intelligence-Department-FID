# Examples

This directory contains example configurations and strategy implementations.

## Contents

- `paper_trading.env` — Minimal environment for paper trading
- `mean_reversion_agent.py` — Example custom strategy agent (educational)

## Running Examples

```bash
# Use the paper trading environment
cp examples/paper_trading.env .env
python -m aiswarm --mode paper
```
